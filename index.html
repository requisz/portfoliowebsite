  • Caloric Bank Web App

    May 2022 - September 2023

    Tech Stack

    UML, Node.js, Express.js, React.js, PostgreSQL, Java, HTML, CSS

    Summary

    The Caloric Bank App is a revolutionary calorie tracking tool based on a unique "banking" concept. Users can "save" calories from one day and "spend" them on another, promoting flexibility in dieting. By adhering to a daily calorie limit, users can opt to consume fewer calories on one day and compensate on the next.

    How it was made

    The development began with the creation of UML Class and Sequence Diagrams for effective planning. We used Node.js to craft server-side components, ensuring optimal performance. Express.js aided us in designing robust RESTful APIs. The user interface, crafted with React.js, delivers a smooth and interactive experience. For data storage and retrieval, PostgreSQL was our choice, and we managed to enhance data retrieval speeds by 30% through query optimizations.

  • Telegram-Discord Crypto Support Bot

    September 2021 - May 2023

    Tech Stack

    Node.js, Express.js, WebSockets, RESTful APIs, MySQL

    Summary

    This bot acts as a bridge between Telegram and Discord, specifically tailored for a Crypto Support Channel. It broadcasts tickets, messages, and other pertinent information from Telegram directly to a Discord server in real-time. This not only enhanced the organization within the group but also expanded its reach, enabling the support channel to assist a broader audience more effectively.

    How it was made

    The foundation of the bot was built using Node.js, a popular runtime for such applications. We utilized Express.js for easy routing and server setup. To achieve the real-time broadcasting capabilities, WebSockets played a crucial role. By integrating with the APIs of both Telegram and Discord, the bot seamlessly transfers information between the two platforms.

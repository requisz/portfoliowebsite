  • Cryptocurrency Customer Support Bot

    September 2021 - May 2023

    Tech Stack

    Node.js, Express.js, WebSockets, RESTful APIs, MySQL

    Summary

    The Cryptocurrency Customer Support Bot is a real-time communication integration tool designed to act as a bridge between Telegram and Discord for a Crypto Support Channel. It broadcasts tickets, messages, and other pertinent information from Telegram directly to a Discord server in real-time. This enhanced organization within the group and expanded its reach, enabling the support channel to assist a broader audience more effectively.

    How it was made

    The bot was built using Node.js, Express.js, WebSockets, RESTful APIs, and MySQL. Node.js served as the runtime for efficient application performance, while Express.js facilitated easy routing and server setup. To achieve real-time broadcasting capabilities, WebSockets played a crucial role. By integrating with the APIs of both Telegram and Discord, the bot seamlessly transfers information between the two platforms, ensuring smooth and effective communication.
